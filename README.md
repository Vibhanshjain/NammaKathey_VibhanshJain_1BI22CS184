# NammaKathey_VibhanshJain_1BI22CS184

Namma Kathey is an Android application designed to preserve and promote the rich cultural heritage, local legends, historical heroes, and inspiring stories of Karnataka in an engaging and interactive way. The application helps children, students, and users explore regional history through storytelling, maps, images, and audio narration.

The project aims to bridge the gap between younger generations and local history by providing a modern digital platform that makes learning culture and history enjoyable and accessible.

---

# ✨ Features

- 📚 Explore stories of Karnataka’s local heroes and legends
- 🗺️ View historical locations using Google Maps integration
- 🔊 Text-to-Speech support for story narration
- ❤️ Save favorite stories for later reading
- 📱 Modern and responsive Material Design UI
- 📖 Category-wise story browsing
- 🔍 Smooth navigation between screens
- 💾 Offline data storage using Room Database
- ⚡ MVVM Architecture for better performance and maintainability

# 🚀 Installation and Setup

## Prerequisites

Make sure the following software is installed:

- Android Studio
- Android SDK
- Kotlin Plugin
- Git

---

# ⚙️ Steps to Run the Project

## 1. Clone the Repository

```bash
git clone https://github.com/Vibhanshjain/NammaKathey_VibhanshJain_1BI22CS184.git
```

---

## 2. Open the Project

- Open Android Studio
- Click on **Open**
- Select the cloned project folder

---

## 3. Sync Gradle

Allow Android Studio to download and sync all required dependencies automatically.

---

## 4. Add Google Maps API Key

Open the `AndroidManifest.xml` file and add your Google Maps API key:

```xml
<meta-data
    android:name="com.google.android.geo.API_KEY"
    android:value="YOUR_API_KEY"/>
```

---

## 5. Run the Application

- Connect an Android device or start an emulator
- Click the **Run ▶️** button in Android Studio
